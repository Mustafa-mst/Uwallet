# Uwallet
Uwallet is a smart wallet application where you can see the money in your account, track your expenses and see investment opportunities.
## Used in the project

 - Firebase
 - Coroutine
 - ViewPager 2
 - Fragment and Navigations
 - Mvvm Design Pattern
 - Retrofit for https request
 - Glide
 - Animations
 - Figma
